# Astram

### Instant Transaction Propagation

- Transactions are broadcast **in real time**
- No waiting for full block completion to relay payments
- Optimized mempool-based network design

### Simple & Clean Design

- Built from scratch with clarity in mind
- No unnecessary features, no bloated logic
- Easy to understand, easy to extend

### Fair Mining Philosophy

- Designed to stay **GPU & CPU friendly**
- ASIC resistance strategies under active research
- Fair participation over hardware domination

### Truly Decentralized

- Peer-to-peer network
- No central authority
- Every node matters
